# 🎉 Welcome to InfoBot!

**InfoBot** is your all-in-one blockchain discovery assistant for Farcaster, Zora, Clanker, Base, and more! Just paste addresses, usernames, or links in chat and get instant information.

## 🚀 Quick Start

**No setup needed!** Just paste anything in chat and InfoBot will automatically detect and respond.

## 📋 Commands

Use these slash commands for specific searches:

• **`/search <query>`** — Universal search (wallets, contracts, profiles, transactions)
• **`/zora <query>`** — Zora accounts, contracts, or creator coins
• **`/clanker <query>`** — Clanker token deployments
• **`/casts <keyword>`** — Search Farcaster casts by keyword
• **`/relay <transaction>`** — Cross-chain transaction details
• **`/help`** — See all commands and features

## 🔍 Auto-Detection (Just Paste!)

InfoBot automatically detects and responds to:

**Addresses & Wallets:**
• `0x...` (Ethereum/Base) — Auto-detects Zora coins, Clanker tokens, Base tokens, Farcaster profiles
• Solana addresses — Auto-detects profiles and tokens

**Farcaster:**
• `@username` — Farcaster profile lookup
• `farcaster.xyz/...` or `warpcast.com/...` — Cast or profile links
• `x.com/...` or `twitter.com/...` — Finds linked Farcaster profiles
• `cast <keyword>` — Search casts by keyword
• `far <keyword>` — Search Farcaster users

**Platforms:**
• `zora.co/...` — Zora profile or coin
• `clanker.world/...` — Clanker token
• `base.org/...` — Base social posts
• `zora <query>` — Zora search
• `wallet 0x...` — Wallet lookup
