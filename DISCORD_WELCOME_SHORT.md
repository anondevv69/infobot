# 🎉 Welcome to InfoBot!

**InfoBot** is your all-in-one blockchain discovery assistant! Just paste addresses, usernames, or links in chat—no commands needed.

## 🚀 Quick Start

**Just paste anything!** InfoBot automatically detects:
• `0x...` addresses → Tokens, wallets, profiles
• `@username` → Farcaster profiles
• `zora.co/...` → Zora profiles/coins
• `clanker.world/...` → Clanker tokens
• `x.com/...` → Farcaster profiles (if linked)
• `cast <keyword>` → Search Farcaster casts
• And more!

## 📋 Commands

• `/search <query>` — Universal search
• `/zora <query>` — Zora-specific search
• `/clanker <query>` — Clanker deployments
• `/casts <keyword>` — Farcaster cast search
• `/relay <tx>` — Cross-chain transactions
• `/help` — Full command list

## ✨ Features

• Multi-page cards with pagination
• Token details with trading links
• Factory detection (Fey, ApeStore, KLIK, Paragraph, etc.)
• Multi-chain support (Base, Ethereum, Solana, and more)
• Farcaster, Zora, and Clanker integration
